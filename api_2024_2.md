# Sobre o projeto

<p><strong>Empresa parceira:</strong>Youtan</p>

<p><strong>Problema:</strong> A empresa parceira vinha enfrentando dificuldades de registro nas avaliações e autoavaliações entre os membros de suas equipes, que eram dispersas geograficamente/fisicamente. Além disso, o número de funcionários aumentava constantemente, o que ... ainda mais a logística.</p>

<p><strong>Solução:</strong> Foi proposto um sistema hierárquico capaz de automatizar essas pesquisas, ... de forma segura e que os níveis de usuários sejam respeitados. </p>
